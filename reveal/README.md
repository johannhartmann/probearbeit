# reveal.js Praesentation

## Dateien

- `index.html` - Reveal Deck
- `slides.md` - Slide-Inhalte

## Lokal starten

Option 1 (Python HTTP Server):

```bash
cd reveal
python3 -m http.server 8080
```

Dann im Browser oeffnen:

- `http://127.0.0.1:8080`

Option 2 (VS Code Live Server o. aehnlich):

- `reveal/index.html` als Webroot starten.
